About This Personal Project

Plannies Mate helps the developers behind Planning Alerts - making planning data more accessible to everyone in Australia.

This suite of tools helps me (Ian), and hopefully you, quickly find which Production scrapers need attention, diagnose issues with existing scrapers, and develop new ones when council websites change.

Remember that this is my personal project so it reflects my own needs. Your klicks (mileage) may vary. See the Planning Alerts How To Write A Scraper and Morph.IO docs page for more information.

Note: Pages derived from private repos require login access. Contact Ian if you are on the planningalerts-scrapers team and don't have access.
